## sane介绍

sane取自'same one',是对团队项目的期许，希望我们做出“风格一样的”项目。

## 内容

该仓库中包含项目开发当中的一些约束，以此来保证项目的一致性。

## 理念

我们致力于通过项目工具来达到目的，凡是能集成到现有框架中的内容，我们都会在工具内植入。
